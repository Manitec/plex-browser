# plex-browser

Lightweight modular browser shell with Plex AI integration.
Built to be forked into purpose-built browsers.

## Stack
- Next.js 14 (App Router)
- Tailwind CSS
- Server-side HTML proxy (rewrites assets, strips CSP)
- Plex AI sidebar (see, observe, ask)
- Bookmarklet observe integration

## Features
- Address bar with back/forward/reload/home
- Smart search (URL vs query detection)
- Multi-engine search (Google, Bing, DDG, Perplexity, Brave)
- Server-side proxy for iframe rendering
- Plex sidebar — ask Plex about any page
- Plex Observe — send selected text or page body to Plex (bookmarklet + native)
- OG metadata preview in Plex panel
- Quick links home screen
